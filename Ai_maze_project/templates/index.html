Algorithm Guide

BFS — Breadth-first. Guaranteed shortest path. Uses a queue.

DFS — Depth-first. Fast but not optimal. Uses a stack.

A* — Optimal + informed. Uses f(n)=g(n)+h(n).

GREEDY — Heuristic only. Fast, not guaranteed optimal.
